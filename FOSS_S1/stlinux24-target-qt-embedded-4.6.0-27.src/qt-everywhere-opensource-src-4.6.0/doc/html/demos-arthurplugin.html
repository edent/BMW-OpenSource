In this demo we demonstrate the abilities of Qt's painting system in combination with Qt Designer's custom widget plugin facilities.

The specialized widgets used in the other demonstrations of the Arthur painting system are provided as custom widgets to be used with Qt Designer. Since each of the widgets provides a set of signals and slots, the effects they show can be controlled by connecting them up to standard input widgets.
